### bullz

It's a minimalist/basic `LALR(1)` parser generator written in C.
It includes a real `LALR(1)` engine with an implementation of (`closure`) algorithms.

It also includes semantic actions with support for passing values ​​using special symbols (`$$, $1, $2, etc.`).
